# Journal Skill — บันทึกการใช้งาน AI

เมื่อผู้ใช้เรียกใช้ skill นี้ ให้ทำตามขั้นตอนต่อไปนี้:

## ขั้นตอน

1. **อ่านไฟล์ `AI_JOURNAL.md`** ที่ root ของโปรเจค เพื่อดูว่ามี Session ถึงเลขอะไรแล้ว
2. **สรุปการสนทนาล่าสุด** ในเซสชันปัจจุบัน — ดูว่าผู้ใช้ถามอะไร AI ตอบอะไร และผู้ใช้ปรับ/แก้อะไรเอง
3. **เพิ่ม entry ใหม่** ต่อท้ายไฟล์ตาม template ด้านล่าง โดยใช้เลข Session ถัดไป
4. **ใช้ภาษาไทย** ในเนื้อหาทั้งหมด

## Template ที่ต้องใช้

ดูไฟล์ [template.md](template.md) สำหรับรูปแบบที่ถูกต้อง และ [examples/sample.md](examples/sample.md) สำหรับตัวอย่างที่เขียนเสร็จแล้ว

## หลักการเขียนสรุป

- **คำถามที่ถาม AI:** ใช้ข้อความจริงของผู้ใช้ หรือสรุปสั้นๆ ถ้ายาวเกิน
- **AI ตอบว่า:** สรุปสาระสำคัญในประโยคเดียว ไม่ต้องคัดลอกทั้งหมด
- **สิ่งที่เราปรับเอง:** ถ้าผู้ใช้ไม่ได้ปรับอะไร ให้เขียนว่า "ยังไม่มีการปรับ" หรือ "-"

## ข้อควรระวัง

- ห้ามเขียนทับเนื้อหาเดิม — ใช้ Edit หรือ append เท่านั้น
- ห้ามลบ Session เก่าออก
- ถ้ายังไม่มีไฟล์ `AI_JOURNAL.md` ให้สร้างใหม่พร้อมหัวข้อ `# บันทึกการใช้งาน AI`
